== Description ==
Bibot will protect your WordPress! No more spam comments and brute-force attacks against user accounts. Lightweight plugin - just few KBs to download and no ads!

Users are required to confirm that they are not a robot before the form can be submitted. It's easy for people and hard for bots.

This WordPress plugin is used by the free [Bibot service](https://bibot.ir) as a third party.

For more information on how it works, refer to [Bibot documents](https://bibot.ir/docs).

You can see our terms of use at [https://bibot.ir/terms_of_use](https://bibot.ir/terms_of_use).



= What is protected with bibot? =
* Comment form
* New password form
* Registration form
* Login form
* Reset password form

== Installation ==
1. Upload bibot files to the "/wp-content/plugins/bibot" directory, or install bibot through the WordPress Plugins page directly.
2. Activate bibot through the WordPress Plugins page.
3. Use the menu Settings => bibot to configure bibot.
4. Get bibot site-key and secret-key from [Bibot service](https://bibot.ir/panel)
